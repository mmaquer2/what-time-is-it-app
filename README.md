# What Time Is It App ?

## What is this?
- this is a simple boilerplate react app to practice devops skills that shows the current date and time

## How to use this?

- the project is built around github actions ci/cd
- github actions ci/cd


## Development and Installation

```bash

# install dependencies
npm i .

# start app
npm run start

# build app
npm run build

# run tests
npm run test

```

## Deployment Methods:

### Surge.sh 

```bash

# install surge
npm i -g surge

# build app
npm run build

# deploy app
surge ./build

```
